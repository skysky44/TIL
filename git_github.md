### Git & GitHub

## CLI vs GUI

  # CLI
    - Command Line Interface
    - 명령 기반의 인터페이스

  # GUI
    - Graphical User Interface
    - 그래픽 기반의 인터페이스

  `인터페이스란 내가 무언가를 조작하는 하나의 접면`

## CLI 명령어 정리
  - ls : 목록
  - mkdir 폴더명 : 디렉토리 생성
  - cd : 이동
  - cd .. : 상위 디렉토리
  - cd . : 현재 디렉토리
  - touch 파일명 : 새로운 파일을 생성
  - rm 파일명 : 파일삭제
  - rm -r 폴더명 : 폴더 삭제 가능(포함한 모든 폴더와 파일 같이 삭제)
  - ~ : 홈 디렉토리
  - pwd : 현재 디렉토리 출력
